# JST PADI

Aplikasi Jaringan Syaraf Tiruan untuk memprediksi hasil produksi padi di kalimantan selatan menggunakan metode backpropagation.

# Frontend
 - Svelte@Sveltekit
 - Tailwindjs
 - DaisyUI
 - GridJS

## Backend

 - FastApi
 - Mysql
 - SQLAlchemy
 - Tensorflow < 2.7.0
